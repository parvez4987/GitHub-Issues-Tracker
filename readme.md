1. What is the difference between var, let, and const?
Answer: Use const for values that won't change and let for variables that needs to change or can be changed. and var is     function-scoped and can be reassigned/ or redeclared.

2. What is the spread operator (...)?
Answer: The spread operator opens an array or object's elements into a new one, making it easy to copy or merge data without changing the original.

3. What is the difference between map(), filter(), and forEach()?
Answer: map() creates a new array by transforming every element, 
        filter() method will return an array of matching elements else will return an empty array if no matching happens.
        forEach() simply executes a function for each item without returning a new array,  It will not return anything, it just applies the conditions to every element. It will not change the original array.

4. What is an arrow function?
Answer: An arrow function is a precise way to write functions using =>, automatically inheriting the "this" value from its surrounding context.

5. What are template literals?
Answer: Template literals are strings wrapped in backticks (`) that let inject variables directly with ${} instead of messing with a million plus signs.
